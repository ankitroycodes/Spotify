# Spotify Clone

This repository contains the code for my Spotify clone website. The project initially started as a UI design exercise, but I decided to code it up and bring it to life! The design features some slight modifications to the original Spotify UI.

## Project Overview

The primary goal of this project was to recreate the Spotify user interface with my own design tweaks. Although it started as a purely visual project, I've decided to take it a step further by implementing some basic functionality. Due to time constraints and my current level of knowledge, the functionality may be limited, but I'll give it my best shot!

## Features

- **Custom UI**: Redesigned Spotify interface with slight modifications.
